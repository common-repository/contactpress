ContactPress is the Responsive Ajax Based Contact Form with User Management.

== Description ==

ContactPress is the Responsive Ajax Based Contact Form with User Management. In User Management it show the List who are all contacted.  It has the option to send a mail to those peoples. Bulk Delete Options for the contacted person and also Bulk Send Mail Options are also available in this plugin.
= ContactPress Features =

 1. It works using the shortcode [contactpress].
 2. Responsive Ajax Based Contact Form.
 3. Manage the Contacted Users.
 4. Mail the Contacted Users.
 5. Mail Option such as Send, Reply, Send All, etc.
 6. Tested in Top Browsers and WordPress Themes.
 7. Simple and Easy to Use.
